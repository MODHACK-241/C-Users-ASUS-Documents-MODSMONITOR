# SystemMonitor-PRO
# SysMonitor Pro > Real-time system monitoring tool with dynamic colored tabs for Windows. ## Features - CPU — cores, frequency, per-core load in real time - RAM — total, used, free, swap - Storage — all disks with usage bars - Network — IP address, traffic, active interfaces - GPU — model, VRAM, driver (NVIDIA, AMD, Intel) - Battery — level, health, estimated time remaining - Screen — resolution, size in inches ## Requirements - Windows 7 / 8 / 10 / 11 - Python 3.8+ (not required for .exe version) ## Installation ```bash pip install psutil rich screeninfo python infos_pc.py ``` ## Build .exe ```bash pip install pyinstaller python -m PyInstaller --onefile --console infos_pc.py ``` ## Usage - Arrow LEFT / RIGHT to switch tabs - CTRL+C to quit ## License MIT License — Free to use and modify..
